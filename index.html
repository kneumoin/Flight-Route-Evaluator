Flight Route Evaluator

оценка маршрутов по YAML-конфигу

О проекте

CLI-инструмент на Go для сравнения заранее заданных маршрутов перелёта из YAML-конфига (не произвольный поиск). Оценивает цену, стыковки, багаж, визовые риски и формирует двуязычный HTML-отчёт.

Быстрый старт

git clone https://github.com/kneumoin/nepal.git
cd nepal
go test ./...
go run ./cmd/flight-routes \
  --config configs/routes.yaml \
  --out ./out \
  --provider mock
open ./out/report.html

Возможности MVP

  • Только ветки из YAML-конфига (one-stop / self-transfer)
  • Mock-режим без API-ключей (--provider mock)
  • Отчёты: HTML (RU/EN), CSV, JSON
  • Провайдеры: mock, Aviasales, Kiwi (stub)
  • Файловый кэш — без базы данных
