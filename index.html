Flight Route Evaluator

Expedition logistics tool for evaluating flight routes to Nepal. Compares predefined routes by price, baggage allowance, connection risk, and visa constraints.

Project status: MVP in development Trekking & mountaineering travel planning
View on GitHub View sample report

Features

  • Compares predefined route branches from config (e.g. MOW → KTM via hub cities)
  • Evaluates baggage allowance, visa risk, and connection timing
  • Generates bilingual HTML reports (RU / EN), plus CSV and JSON
  • Integrates flight pricing APIs (Travelpayouts / Aviasales, Kiwi) with filesystem cache
  • Mock mode for offline development without API keys

Purpose

We are building a route evaluation tool for expedition logistics — trekking and mountaineering travel to Nepal. The tool compares predefined routes by price, baggage allowance, connection risk, and visa constraints.

Travelpayouts API is used for flight availability and pricing data. This is an analytics and planning tool, not a ticket shop.

Quick start

git clone https://github.com/kneumoin/Flight-Route-Evaluator.git
cd Flight-Route-Evaluator
go run ./cmd/flight-routes \
  --config configs/routes.yaml \
  --out ./out \
  --provider mock
open ./out/report.html
